CV学习笔记（0）入坑篇

发表于 2018-05-20 | 分类于 计算机视觉 | 阅读次数

以下内容整理自 知乎Live-计算机视觉从入门到放肆, 也算是给像我一下的CV小白一点入门的参考：

什么是图像和视频

图像

一张图片包含了维数、高度、宽度、深度、通道数、颜色格式、数据首地址、结束地址、数据量等。此外还有图像深度、图像压缩格式。

视频

视频包含码率、帧率、分辨率、清晰度、带宽。原始视频 = 图片序列。

压缩后的视频会采取各种算法减少数据的容量。 其中IPB就是最常见的。I帧：关键帧， P帧：跟上一个关键帧的差别，B帧：跟上一个和下一个关键帧的差别

CPU& GPU

学习计算机视觉要对CPU和GPU的工作原理有一定了解，要会使用GPU进行计算。

和其他学科的关系

计算机视觉|数字图像处理|机器视觉 的区别
  • 计算机视觉： 软件、算法层面对图像的处理、理解
  • 数字图像处理：像素级别对图像的处理和转换
  • 机器视觉：跟硬件紧密结合，例如机器人等

需要的能力

  • Python（and numpy）， C/C++
  • 数学基础，微积分/线性代数/概率论
  • 机器学习

参考书

  • 入门级：Computer Vision: Models, Learning and Inference. Written by Simon J.D. Prince.
  • 已经有基础，想学习OpenCV，重实践，轻理论：Learning OpenCV.
  • 要求高，希望对CV有深入了解：Computer Vision: Algorithms and Applications. Written by Richard Szeliske.
  • 更多的几何知识，3D，VR，AR: Multiple View Geometry in Computer Vision. Written by Richard Harley.

公开课

  • Stanford CS 131 Computer Vision: Foundations and Applications
  • Stanford CS231A Computer Vision: from 3D reconstruction to recognition
  • Stanford CS231N: Convolutional Neural Networks for Visual Recognition
  • Stanford CS 223B

网站

收录CV领域最新的研究成果，并持续更新数十年：www.visionbib.com/bibliography/contents.html

深度学习书籍

Goodfellow and Yoshua Bengio

开源软件

  • OpenCV：计算机视觉库 快速实现经典算法
  • TensorFlow：开源机器学习深度学习框架
  • Caffe：深度学习卷积神经网络开源框架
  • ffmpeg：强大的视频处理工具

阅读文献

  1. 先熟悉所在方向的发展历程，阅读里程碑文献：
    1. 目标检测：RCNN， FastRCNN, Faster RCNN, SPPNET, SSD, YOLO.
    2. 目标跟踪：DLT, SO-DLT.
    3. 对抗网络：GAN，CGAN，DCGAN，LAPGAN.
  2. 每周看一下别人最新的工作：arxiv.org/list/cs.CV/recent
  3. 阅读论文时搞明白（论文笔记）：
    1. 解决了什么问题
    2. 用了什么方法
    3. 效果如何
    4. 还存在什么问题
